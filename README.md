
# Project Title

EatWell_Resturant_WebSite_Backend


## Run Locally

Clone the project

```bash
  git clone https://github.com/Arindam01gi/EatWell-Website.git
```

Go to the project directory

```bash
  cd EatWell-Website 
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
